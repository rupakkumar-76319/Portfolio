E-Commerce Sales Analytics Dashboard

Power BI dashboard for analyzing sales performance, customer behavior, and product trends using 541K+ records.

📌 Overview

This project presents an end-to-end business intelligence solution built using Power BI and SQL on a large-scale e-commerce dataset containing over 541,000 records.

The objective was to analyze sales performance, customer behavior, product trends, and time-based growth patterns to derive actionable business insights.

Advanced data modeling and DAX calculations were implemented to compute key performance indicators such as Month-over-Month (MoM) growth, Year-over-Year (YoY) growth, Average Order Value (AOV), and revenue contribution metrics.

The final dashboard enables interactive exploration of data through drill-down visualizations, helping stakeholders make data-driven decisions effectively.
