Weather Cast ML System

End-to-end Machine Learning system for rainfall prediction with deployed API.

📌 Overview

The Weather Cast ML System is an end-to-end machine learning application designed to predict whether it will rain the next day using historical weather data.

The project covers the complete ML lifecycle including data preprocessing, feature engineering, model training, evaluation, and deployment via a REST API.

It is built with a strong focus on real-world usability, allowing users to input weather parameters and receive instant predictions through a deployed interface.

⚙️ Tech Stack

Python Scikit-learn FastAPI XGBoost Pandas

🔄 Pipeline

Data → Preprocessing → Model → Deploy

🧠 How Prediction Works

The system takes multiple weather-related inputs such as temperature, humidity, wind speed, and pressure.

  • Data is preprocessed (handling missing values, encoding categorical features)
  • Features are scaled and transformed
  • Multiple ML models are trained and compared
  • Best model (based on accuracy & performance) is selected
  • Final prediction is returned via FastAPI endpoint
